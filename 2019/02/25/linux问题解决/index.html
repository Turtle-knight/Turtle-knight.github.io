PS：先将BIOS中的secure boot设置为 disable

  5. 重启之后就会看见系统选择界面

2.双系统下给Ubuntu18.04磁盘扩容，并挂载分区

  1. 在Windows上分出一块空白区。打开磁盘管理，选择自己要分割的分区，压缩卷，输入大小，新建卷。

  2. 在Ubuntu下已root身份登入，终端输入

    显示如下，找到刚创建的分区，这里假设为 /dev/sda12。

继续输入：sudo umount /dev/sda12 解除绑定。

接着输入: mkfs -t ext4 /dev/sda12 进行格式化

这个命令是将该分区格式化为 ext4文件系统。格式化成功后显示该分区的UUID，这里记录下UUID。

在home下创建一个新的目录，此目录为新分区挂载的目录。这里我新建了一个/home/new目录。

最后，以root身份编辑/etc/fstab文件，。

在fstab文件最后一行添加新分区的UUID，方便以后开机自动挂载。

保存重启后，新分区会显示已被挂载到目录/home/nwe下。

以上第二步的操作大部分可以在图形化界面完成，在搜索中输入“磁盘”。
