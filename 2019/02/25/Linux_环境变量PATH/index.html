环境变量$PATH

在Linux中，执行命令时，系统会按照PATH的设置，去每个PATH定义的路径下搜索执行文件，先搜索到的文件先执行。（From：《鸟哥的LINUX私房菜》）

改变PATH

直接修改$PATH值：

  • 生效方法：立即生效
  • 有效期限：临时改变，只对当前终端有效，关闭后恢复原来的PATH
  • 用户局限：仅当前用户

  • 生效方法：系统重启
